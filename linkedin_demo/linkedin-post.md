# LinkedIn Post Caption

I built a full-stack AI job recommendation website for students, freshers, interns, and experienced candidates looking for relevant jobs in India and abroad remote markets.

What it does:

- Accepts resume upload or pasted resume text
- Extracts skills, experience, projects, certifications, tools, education, and suitable job roles
- Collects preferences like candidate level, years of experience, job type, location, expected salary, relocation, and preferred roles
- Recommends matching jobs with resume match percentage, required skills, experience fit, location, source, and direct apply/search links
- Applies strict filters so freshers do not see senior jobs, interns only see internships, and experienced candidates do not see roles above their experience level
- Includes India + abroad remote application links from sources like LinkedIn, Naukri, Indeed, Wellfound, We Work Remotely, Remotive, Turing, Arc, and company career pages
- Gives resume improvement suggestions for missing skills, keywords, projects, certifications, and ATS readability

This was a useful full-stack project because it combines resume parsing, deterministic matching logic, strict eligibility filters, ranking, generated application links, and a responsive dashboard UI.

Tech stack:
FastAPI, Python, React, TypeScript, Vite, localStorage, responsive CSS

Planning to keep improving it with live job APIs, login, saved profiles, and smarter resume scoring.
