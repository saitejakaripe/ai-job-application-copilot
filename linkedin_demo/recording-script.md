# LinkedIn Demo Recording Script

Use this for a short 35-45 second screen recording.

## Setup

1. Open the app: http://127.0.0.1:5173/
2. Start a screen recording with QuickTime Player:
   - File -> New Screen Recording
   - Select the browser window or app area
   - Start recording

## Demo Flow

1. Show the empty dashboard.
   - Say or imply: "This is a resume-aware job recommendation app."

2. Paste a sample resume.
   - Use a software/AI developer resume snippet.
   - Click "Analyze resume".

3. Show extracted profile details.
   - Skills
   - Suitable roles
   - Experience
   - Tools/projects

4. Set preferences.
   - Candidate level: Experienced
   - Years: 3
   - Job types: Full-time + Remote
   - Location: Remote
   - Roles: AI Engineer, Full-Stack Developer
   - Market: Both India and abroad remote jobs

5. Click "Find matching jobs".
   - Pause on the result count.
   - Show job cards with match percentage, reason, skills, source, save, and apply.

6. Scroll to resume improvements and application links.
   - Show missing skills, keywords, projects, certifications, ATS tips.
   - Show application links grouped by LinkedIn, Naukri, Indeed, Wellfound, Remotive, Turing, Arc, etc.

7. End on the dashboard.
   - Stop recording.

## Suggested Video Length

Keep it under 45 seconds for LinkedIn.
